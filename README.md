**Fixed Cropper**

**Fixed Cropper** は、
固定サイズのキャンバスに画像を配置し、

SNS投稿・サムネイル作成・比率調整などに使える

**Windows 向け画像切り抜きツール**です。

Python + PySide6 で開発され、

**exe ファイルとして配布**されます。

\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_

**特徴**

•固定サイズキャンバス（1080×1920 / 1920×1080 など）

•画像のドラッグ＆ドロップ対応

•画像の移動・拡大縮小（マウス／キーボード）

•画像倍率の数値表示・直接入力（%）

•背景色の指定・履歴保存

•カスタムキャンバスサイズの保存・再利用

•書き出しプレビュー表示

•操作説明・ショートカットをオーバーレイ表示

\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_

**動作環境**

•Windows 10 / 11

•Python のインストールは不要（exe版）

\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_

**インストール（exe版）**

**1. ダウンロード**

GitHub の Releases ページから

最新版の exe ファイルをダウンロードしてください。

例：FixedCropper\_v1.1.0.exe

**2. 起動**

•ダウンロードした exe を ダブルクリックするだけで起動します

•インストール作業は不要です

※ 初回起動時、Windows の警告が出る場合があります

（個人開発アプリのため）

\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_

**基本的な使い方**

**画像を開く**

•**ファイル → 画像を開く**

•または画像ファイルをウィンドウへドラッグ＆ドロップ

\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_

**画像の移動**

•**矢印キー**：画像を移動

oShift + 矢印キー：微調整

•**マウスドラッグ**：画像を移動

oShift + ドラッグ：平行移動

\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_

**画像の拡大・縮小**

•**Ctrl + ホイール**：ズーム

•**Ctrl + Shift + ホイール**：微調整

•右下オーバーレイの ＋ / − ボタン

•画像倍率欄に 数値を直接入力（%）

※

•**画像上**：画像自体をズーム

•**背景上**：表示倍率（ビュー）をズーム

\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_

**書き出し**

•**ファイル → 書き出し**

•現在のキャンバス状態を画像として保存します

\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_

**保存される設定**

以下は自動的に保存・復元されます：

•最後に開いたフォルダ

•カスタムキャンバスサイズの履歴

•背景色の履歴
